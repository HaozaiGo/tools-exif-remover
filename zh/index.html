EXIF Remover 是什么？

一个用于查看和清除照片 EXIF 元数据的隐私工具，可识别 GPS 位置、相机型号、拍摄日期等信息。

主要功能

  • Read EXIF metadata
  • Detect GPS location data
  • Warn about privacy-sensitive metadata
  • Remove embedded metadata
  • Compare file size before and after cleanup
  • Process images locally in the browser

适合场景

  • 开发、调试或发布前的日常处理
  • 在浏览器端快速完成任务，减少数据上传
  • 为技术文档、测试数据或线上内容做准备

常见问题

什么是 EXIF 数据？

EXIF 是图片内嵌元数据，可能包含相机型号、拍摄时间、曝光参数和 GPS 坐标。

为什么要删除 EXIF？

EXIF 可能泄露位置等隐私信息，分享图片前清除可以降低隐私风险。

删除 EXIF 会影响画质吗？

清除过程可能重新编码图片，工具会尽量保持较高画质。

图片会上传吗？

不会。查看和清除元数据都在浏览器端完成。
